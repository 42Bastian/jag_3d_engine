# Atari Jaguar 3D Engine

This is a new 3D engine for the Atari Jaguar (or at least a try). 

Written in 100% GPU assembly.

The demo uses Hively-player which runs completely in the DSP.

## Cybermorp (CM) shapes

Recompiled from binary blob. See

https://forums.atariage.com/topic/388467-cybermorph-asset-decompile/

## Building

Use latest lyxass, which support condtion combinations like 'jr pl_ne'.

https://github.com/42Bastian/lyxass
